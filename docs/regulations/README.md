# Regulations Watchdog

This directory contains reports generated by the **Ultracite Regulation Watchdog**.

## Overview

The Watchdog runs periodically (nightly via GitHub Actions) to check for changes in:
- HS Classification Rules
- FDA Prior Notice Requirements
- Labeling Requirements

## Reports

Reports are generated in the `reports/` directory with a timestamp.
Snapshots of the last known state are stored in `snapshots/`.

## Configuration

Queries are defined in `backend/app/watchdog/queries.yaml`.
